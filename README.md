## 🌟 Особенности (Features)

- 🎓 **Три роли пользователей**: Студенты, Преподаватели (Авторы) и Администраторы.
- 🎮 **Интерактивные мини-игры**: Встроенные мини-игры ("Побег робота", "2D Майнкрафт") для обучения основам программирования.
- 🛠️ **Конструкторы уровней**: Полноценные визуальные редакторы в админ-панели для создания уровней к мини-играм.
- 📱 **Progressive Web App (PWA)**: Возможность установки платформы на мобильные устройства и планшеты. Приложение работает быстро и плавно в полноэкранном режиме.
- 🎨 **Современный UI/UX**: Стеклянный морфизм (Glassmorphism), неоновые эффекты, полностью темная тема "Ocean-Blue".
- 📈 **Кабинеты и аналитика**: Личные кабинеты для отслеживания прогресса студентов, рейтингов и статистики курсов.

## 🏗️ Архитектура (Architecture)

Проект использует архитектуру монорепозитория (через **Turborepo**), разделяя код на независимые, масштабируемые микро-пакеты.

### Стек технологий:
- **Фронтенд**: Next.js 15 (App Router), React, Tailwind CSS, Lucide Icons.
- **Бэкенд**: NestJS, TypeScript, REST API.
- **База Данных**: PostgreSQL с использованием Prisma ORM.
- **Инфраструктура**: Docker, PM2, Nginx, Turborepo.

### Структура монорепозитория:
```
amazing-pascal/
├── apps/
│   ├── web/        # Frontend на Next.js (пользовательский интерфейс)
│   └── api/        # Backend на NestJS (бизнес-логика)
├── packages/
│   ├── database/   # Схемы Prisma и клиенты для работы с БД
│   ├── shared/     # Общие типы, утилиты и константы
│   └── tsconfig/   # Общие конфигурации TypeScript
```

## 🚀 Установка и запуск (Installation & Setup)

Для запуска проекта локально вам понадобятся: **Node.js 20+**, **pnpm**, и **PostgreSQL** (или Docker для запуска БД).

### 1. Клонирование репозитория
```bash
git clone https://github.com/Lyman-Frank/LearnHub.git
cd LearnHub
```

### 2. Установка зависимостей
```bash
pnpm install
```

### 3. Настройка окружения
Скопируйте пример файла конфигурации:
```bash
cp .env.example .env
```
Убедитесь, что в файле `.env` указана правильная строка подключения `DATABASE_URL` к вашей базе данных PostgreSQL.

### 4. Настройка базы данных
Генерация клиента Prisma и применение миграций:
```bash
pnpm --filter @repo/database run generate
pnpm --filter @repo/database run push
```
*(Опционально: запуск `pnpm --filter @repo/database run seed` для наполнения базы тестовыми данными)*.

### 5. Запуск серверов разработчика
```bash
pnpm run dev
```
- **Web (Next.js)** будет доступен по адресу: `http://localhost:3000`
- **API (NestJS)** будет доступен по адресу: `http://localhost:3001`

## 📱 Progressive Web App (PWA)

Платформа полностью поддерживает PWA. Чтобы установить ее на устройство:
- **На мобильных устройствах (iOS/Android)**: Зайдите на сайт через Chrome или Safari, нажмите кнопку **"Установить приложение"** (Install App) в шапке профиля или выберите в меню браузера "Добавить на главный экран".
- **На десктопе**: В адресной строке Chrome появится иконка скачивания.
